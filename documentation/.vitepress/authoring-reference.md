# Справочник оформления документации (Vue Inspector)

Краткие подсказки и примеры для `*.md` в каталоге `documentation/`. Синтаксис совместим с **VitePress** (на базе Markdown + расширения).

---

## Заголовки

Используй иерархию: один `#` на страницу (заголовок в frontmatter дублируют `title`).

```markdown
# Уровень 1 — только один на файл (опционально дубли title)
## Уровень 2 — разделы
### Уровень 3 — подразделы
#### Уровень 4
```

---

## Выделение текста

| Синтаксис | Результат |
|-----------|-----------|
| `**жирный**` | **жирный** |
| `*курсив*` | *курсив* |
| `***жирный курсив***` | ***жирный курсив*** |
| `` `код` `` | `код` |
| `~~зачёркнутый~~` | ~~зачёркнутый~~ |

---

## Списки

Ненумерованный:

```markdown
- пункт
- вложенный
  - вложенность 2 пробела
```

Нумерованный:

```markdown
1. шаг
2. шаг
```

Задачи (если включено в теме):

```markdown
- [x] сделано
- [ ] todo
```

## Код: блоки

Указывай язык для подсветки:

````markdown
```ts
const x: number = 1
```
````

````markdown
```vue
<template>
  <div />
</template>
```
````

**Фокус на строках** (VitePress):

````markdown
```ts {2,4-6}
line 1
line 2  // выделена
line 3
line 4
```
````

**Номера строк** (`ts:line-numbers`):

````markdown
```ts:line-numbers
const a = 1
```
````

## Кастомные блоки (VitePress)

```markdown
::: tip Подсказка
Краткая полезная заметка.
:::

::: info Информация
Нейтральные сведения.
:::

::: warning Внимание
Предупреждение о последствиях.
:::

::: danger Опасно
Риски или разрушительные действия.
:::

::: details Нажми, чтобы раскрыть
Скрытый текст.
:::
```

## Frontmatter (YAML в начале файла)

```yaml
---
title: Заголовок страницы в браузере
description: Краткое описание для SEO
outline: deep
---
```

`outline: false` — скрыть оглавление справа; `deep` — глубже уровни заголовков.

## Краткий чеклист перед публикацией

1. Один `title` в frontmatter или один `#` в тексте.
2. Внутренние ссылки — пути вида `/section/page` без `.html`.
3. Код-блоки с языком.
4. Для предупреждений — `::: warning` или `::: danger`.
5. Скриншоты — в `documentation/public/` или относительные пути к ассетам.
