## [2.4.0] - 2026-04-19

### Network

* **Network** table uses virtualization for speed and fewer UI freezes.
* Request detail opens on **Response** by default.
* **Breakpoint** editing auto-opens the **Request** tab for the paused method.
* With **Pause**, inspector interceptors detach — native DevTools show the **original initiator** instead of `injected.ts`.

---

### Props

* Major Props performance — **auto refresh** is safer to use.
* **Props** column split into **Passed** and **Declared** (all columns can be hidden):

  * **Passed** — props currently received;
  * **Declared** — props declared on the component.
* **Details Props** adds **Size**.

#### Inspect (Props Inspector)

* Page element picking similar to **Chrome DevTools Elements**.
* Pick on-page to filter the **Props** list quickly.
* Highlights Vue components on hover.
* Tooltip: **Name**, **Root element**, **Received Props**, **Nested components**.
* After pick, **Props** shows the selected subtree only.
* **Filtered** badge clears the filter.

---

### UI / UX

#### Windowed mode

* Windowed layout with remembered position for both launch modes.
* Chevron/drag handle refresh for iframe expand.
* Dock the pill to any browser edge with auto-stretch along that edge.

#### Auto launch

* **Whitelist** / **Blacklist** site lists for automatic overlay launch.
* Extension **popup** shortcuts:

  * **Forced Launch** — ignore whitelist/blacklist and static-site guard;
  * **Reset Settings** — wipe all extension settings.

#### Other

* JSON styling fixes for very large payloads.
* Exact search when wrapping the query in quotes; without quotes, **like** substring search.
* Small visual tweaks.
* **Esc** in **DevTools** mode: first closes nested tabs; when nothing left, focuses **Console**.

---

### Performance

* Faster Props and Store discovery; higher read limits for Props.
* Refactor: removed unused modules, dead code, duplicated logic.
