# Введение

Vue Inspector — инструмент для разработчиков и QA для приложений на Vue 3. С его помощью можно отлаживать компоненты, менять состояние Pinia и подменять ответы API прямо в браузере — без прокси, пересборки и обходных путей.

## Дальнейшие шаги

- Установите расширение для браузера или используйте bookmarklet со [стартовой страницы](../../index.html) (корень сайта, рядом с каталогом документации).
- Откройте DevTools и запустите Vue Inspector на своём Vue-приложении.
