# JSON Mode (режим просмотра JSON)

В **Options → General** переключатель **JSON Mode** задаёт реализацию **просмотра и правки JSON** во всём инспекторе (тела запросов и ответов во вкладке **Network**, тела в **Mock response**, снимки **Pinia**, данные компонента на **Props**, если содержимое распознаётся как JSON).

Тела **не в формате JSON** (XML, обычный текст и т.д.) по-прежнему открываются в **одном текстовом виде** с подсветкой Prism, где это поддерживается; переключатель в первую очередь влияет на работу с **JSON**.

---

## Classic

**Classic** — встроенный **текстовый редактор с подсветкой Prism** (`JsonTextEditor`):

- Одна **текстовая** область — **отдельного древовидного интерфейса нет**.
- В режиме **только чтение** включается подсветка JSON (и других поддерживаемых языков); в **редактировании** по-прежнему текст с проверкой корректности JSON.
- Учтены **большие фрагменты**: после порога размера подсветка ослабевается, чтобы не подвешивать интерфейс; для очень больших **не-JSON** полей только чтения может использоваться облегчённый просмотрщик.

Выбирайте Classic, если нужен **минимальный и быстрый текстовый** режим.

---

## Advanced

**Advanced** — **`vanilla-jsoneditor`** (компонент `JsonEditor` в интерфейсе):

- **Два представления одного и того же JSON**: **Text** (код) и **Tree** (структура). Переключение — на панели редактора; последний выбор (**Text** или **Tree**) **сохраняется** в настройках (`json.mode`).
- **Режим Text**: инструменты форматирования (например **Format** / **Compact**, где разрешено редактирование), **Undo / Redo**, **Поиск**, копирование и прочие возможности библиотеки.
- **Режим Tree**: свёртка/развёртка узлов, **хлебные крошки / путь** в документе, контекстные меню по узлам, а также связанные с деревом сценарии сортировки и правки, которые добавляет приложение.

Выбирайте Advanced, если нужны **дерево**, **история правок**, **поиск** и более богатое редактирование JSON.

---

## Сводка

| | **Classic** | **Advanced** |
|---|-------------|--------------|
| **Движок** | Текст + Prism (`JsonTextEditor`) | vanilla-jsoneditor |
| **Дерево** | Нет | Да (Text + Tree) |
| **Когда удобно** | Лёгкий текст и подсветка | Структурное редактирование и панель инструментов |
