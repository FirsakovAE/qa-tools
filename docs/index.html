Vue Inspector

Инструмент инспекции Vue/Pinia

Этот инструмент позволяет инспектировать Vue компоненты и Pinia хранилища на страницах. Доступно два режима использования: как расширение браузера или как standalone-скрипт.

Последнее обновление
Как использовать
1

Перетащите кнопку ниже в панель закладок браузера

2

Откройте страницу с Vue приложением

3

Кликните по закладке — inspector появится внизу страницы

Vue Inspector ← Перетащите в закладки
Режим работы

Standalone Mode

В standalone режиме некоторые функции недоступны: инспекция других вкладок, фоновая синхронизация. Все основные возможности инспекции Vue и Pinia работают.

Установка расширения
1

Скачайте последнюю версию расширения

2

Распакуйте архив в удобное место

3

Откройте chrome://extensions/ в браузере

4

Включите "Developer mode" (режим разработчика) в правом верхнем углу

5

Нажмите "Load unpacked" и выберите папку dist/ из распакованного архива
